以下統一使用「左閉右閉」區間

之前本來慣用「左閉右開」，但發現在某些情況下，會需要特別向上取整，或是給出不同返回值，太瑣碎了，乾脆換一套新的，更容易理解。

TL;DR

第一組:

返回值 <= target

返回值 > target

第二組:

返回值 < target

返回值 >= target

這些都是不是通靈，是一步一步推論後整理出來，方便記憶的結論！

細節討論

通常有【相等返回】，和四個【一般情況】

  • 相等返回: 若找到 nums[mid] == target 就返回 mid
  • 一般情況
    1. 返回值 >= target
    2. 返回值 > target
    3. 返回值 <= target
    4. 返回值 < target
