3. BrownieеҢҜе…Ҙgithubзҡ„еӨ–йғЁеҗҲзҙ„пјҢcompileеҫҢеҮәзҸҫе ұйҢҜ

йҰ–е…ҲжҳҜеӣ зӮәеҢҜе…Ҙзҡ„pragmaзүҲжң¬е•ҸйЎҢпјҢйңҖиҰҒзўәе®ҡйҖҷеҖӢеӨ–йғЁеҗҲзҙ„дҪҝз”Ёзҡ„жҳҜд»ҖйәјзүҲжң¬зҡ„solidity

жҲ‘иҮӘе·ұеҮәйҢҜзҡ„еҺҹеӣ еҸҜиғҪжҳҜеӣ зӮәйҖҷй …:

When the compiler version is not explicitly declared, Brownie looks at the version pragma of each contract and uses the latest matching compiler version that has been installed.

жҲ‘жң¬ең°зҡ„йӣ»и…Ұе·Із¶“е®үиЈқдәҶжңҖж–°зҡ„v0.8.11пјҢ

дҪҶжҳҜеҳ—и©ҰйҒҺеҫҢзҷјзҸҫпјҢдјјд№Һз„Ўжі•еҲ©з”Ёе•ҸйЎҢ1зҡ„и§Јжі•(жүӢеӢ•ж”№и®Ҡз·ЁиӯҜзүҲжң¬)дҫҶи§Јжұә

зӣ®еүҚжҺЁжё¬жҳҜеӣ зӮәеңЁгҖҢжІ’жңүеңЁbrownie-config.yamlдёӯжҢҮе®ҡзүҲжң¬гҖҚзҡ„жғ…жіҒдёӢпјҢBrownieжңғзӣҙжҺҘйҒёеҸ–е·Іе®үиЈқзҡ„жңҖж–°з·ЁиӯҜеҷЁпјҢд№ҹе°ұжҳҜжҲ‘зҡ„v0.8.11дҫҶйҖІиЎҢз·ЁиӯҜ

и§ЈжұәиҫҰжі•: brownie-config.yamlдёӯиЁӯе®ҡзүҲжң¬

е…¶д»–еӯёеҲ°зҡ„зҹҘиӯҳй»һ:

  1. BrownieеҸҜд»Ҙиҷ•зҗҶеӨҡзЁ®зүҲжң¬зҡ„importпјҢsolidityдёҚиЎҢ

    • д»Ҙv0.6з·ЁиӯҜзҡ„еҗҲзҙ„пјҢеҰӮжһңimportйҖІv0.8пјҢsolidityжңғе ұйҢҜ

    • д»Ҙv0.6з·ЁиӯҜзҡ„еҗҲзҙ„пјҢimportйҖІд»Ҙv0.8з·ЁиӯҜзҡ„еҗҲзҙ„пјҢsolidityдёҚжңғе ұйҢҜ

  2. Brownieжңғе…ҲеңЁinterfaceзҡ„pragma configдёӯжүҫеҲ°жңҖж–°зҡ„зүҲжң¬пјҢе„Әе…Ҳй ҶеәҸжҳҜеңЁbrownie-config.yamlд№ӢеүҚзҡ„пјҢжүҖд»Ҙжң¬дҫҶзҡ„brownie-config.yamlиЁӯзҪ®еҸҜиғҪжңғиў«иҰҶи“Ӣ

  3. еҗҢжҷӮпјҢcommentиө·дҫҶзҡ„pragma configйӮ„жҳҜжңғиў«з·ЁиӯҜпјҢйҖҷжҳҜBrownieзҡ„bug

  4. еҰӮжһңеңЁbrownie-config.yamlдёӯиЁӯе®ҡжҳҺзўәзҡ„зүҲжң¬пјҢж•ҙеҖӢеҗҲзҙ„йғҪжңғз”ЁйӮЈеҖӢзүҲжң¬дҫҶз·ЁиӯҜ
